\begin{abstract}
	\textit{This report presents the design, implementation, and evaluation of an autonomous mobile robot integrated with the Robot Operating System (ROS). The project shows how a given task containing mapping, localization, navigation, motor control and perception using camera algorithms can be solved by the use of ROS. At the end this report will be having a conclusion based on the results from testing the robot on the demo day, discussing alternatives of tools/algorithms for a better performance against predefined objectives and suggesting areas for further improvement.}
\end{abstract}
